# MLOPS__MIPT
Здесь выполняю учебные проекты МФТИ по развертыванию микросервисов

## Домашнее задание 1. Знакомство с API
Задание выполняется в рамках модуля 1 «Проектирование API». Формат выполнения — индивидуально. Работать необходимо в облачном Google Colab с использованием библиотек FastAPI, Uvicorn. Форма сдачи — ссылка на ноутбук в Google Colab с кодом, комментариями и визуализациями. Сдается в форму сдачи, количество попыток сдачи — 1.

__Цели задания__
Задание направлено на отработку базовых навыков проектирования API и закрепление материала модуля 1.

После выполнения студент:
  - освоит базовые подходы (code-first/API-first)
  - закрепит навыки написания простого сервера с автодокументацией
  - оформит результаты в виде структурированного ноутбука с кодом, визуализациями и комментариями
